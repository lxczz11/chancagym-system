ChancaGym - Sistema de Gestión de Gimnasio 🏋️‍♂️
Este proyecto es una solución integral para la administración de gimnasios, desarrollada como parte del curso de Ingeniería de Sistemas. El sistema permite gestionar desde el registro de clientes hasta el control de membresías, optimizando la operatividad del negocio.

🚀 Características Principales
Gestión de Socios: Registro, edición y seguimiento de clientes.

Control de Membresías: Monitoreo de fechas de vencimiento y tipos de planes.

Interfaz Responsiva: Diseño adaptado para diferentes dispositivos (Desktop/Tablet).

Seguridad: Control de acceso básico para administradores.

🛠️ Tecnologías Utilizadas
Frontend: HTML5, CSS3 y el framework Materialize para un diseño moderno.

Lógica: JavaScript (ES6+).

Backend: Node js , Mysql. 

📋 Metodología de Trabajo (Ingeniería de Software)
Para este proyecto se aplicaron principios de Metodologías Ágiles (Scrum), asegurando una entrega por etapas:

Product Backlog: Definición de necesidades del gimnasio.

Sprint Planning: Priorización de funcionalidades críticas como el registro de usuarios.

User Stories: Redacción de historias de usuario para entender las necesidades del recepcionista y el dueño del local.
